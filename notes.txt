Проект - портфель инвестора.

1 Расчет состава портфеля
1.1 Выгрузка из api криптобиржи состава портфеля. Активы портфеля распределяются по группам. Группы задаются пользователем.
1.2 Запрос из api криптобиржи текущего курса криптовалют в портфеле. Запрос по расписанию, кешируем в Redis на час.
1.3 Имея в базе список позиций и актуальные курсы можно рассчитать соотношение в % разных групп активов в портфеле.

2 Расчет доходности
2.1 Выгрузка из API криптобиржи истории транзакций в Postgress. По timastamp последней транзакции в базе смотрим за какой период догрузить транзакции.
2.2 Фильтрация транзакций на пополнение аккаунта. Расчет суммы вложенных денег за выбранный период.
2.3 Сравнивая сумму вложенного и текущую стоимость получаем текущую доходность.
2.4 Добавляем этот расчёт в cron, чтобы собирать статистику этого показателя и сравнивать по периодам.
